# Installed Apps

> Тестовое задание для компании **Доктор Веб**

Приложение для просмотра информации об установленных приложениях на устройстве.

---

## Функционал

- Список всех установленных приложений
- Детальная информация о приложении:
  - Название
  - Версия
  - Имя пакета
  - MD5-контрольная сумма APK
- Запуск выбранного приложения

---

## Архитектура

**MVI (Model-View-Intent)**

| Компонент | Описание |
|-----------|----------|
| `State` | Состояние экрана |
| `Intent` | Намерения пользователя |
| `Action` | Одноразовые события (навигация, запуск) |

---

## Структура проекта

```
com.android.installedapps/
├── core/
│   ├── data/          # Реализация репозиториев
│   ├── di/            # Koin модули
│   ├── domain/        # Модели и интерфейсы
│   ├── navigation/    # Навигация
│   └── ui/            # UI утилиты
└── feature/
    ├── apps_list/     # Экран списка приложений
    └── app_detail/    # Экран деталей приложения
```

---

## Стек технологий

| Технология | Назначение |
|------------|------------|
| Jetpack Compose | UI |
| Koin | Dependency Injection |
| Navigation Compose | Навигация |
| ViewModel + MVI | Архитектура |
| MD5 | Контрольная сумма (128 бит) |

---

## Требования

- **Min SDK:** 24 (Android 7.0)
- **Target SDK:** 36
